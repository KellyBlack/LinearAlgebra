\begin{frame}{Row Operation}

  \begin{itemize}
  \item Multiply each entry in a row by a constant
  \item Add a multiple of one row to another
  \item Swap two rows
  \end{itemize}
  
\end{frame}

\begin{frame}{Row Echelon Form}

  \begin{itemize}
  \item Our goal is to solve a linear system.
  \item Use the top row to make all zeros in the coefficients in the
    rows below the first column without all zeros. (``Leading coefficients'')
  \item Move down to the next row.
  \item Keep repeating until you get to the last row.
  \end{itemize}

  \uncover<2->{
    Why?
    \begin{itemize}
    \item Solve last equation for one variable
    \item Move to the row above and solve for one variable.
    \item Keep repeating until you get to the first row.
    \end{itemize}
  }

  
\end{frame}

\begin{frame}{Row Echelon Form}

  When do you stop? When the augmented matrix is in row echelon form:
  \begin{itemize}
  \item Any row with all zeroes is at the bottom of the augmented
    matrix.
  \item For a given row, the first non-zero entry is to the right of
    all the columns where the first non-zero entry is in all of the
    rows above it.
  \item For a given row, all of the entries below the first non-zero
    entry are zero.
  \end{itemize}
  
\end{frame}
